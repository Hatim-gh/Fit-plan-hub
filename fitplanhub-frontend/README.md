# FitPlanHub Frontend

React frontend for FitPlanHub fitness plans platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Make sure the backend is running on `http://localhost:5000`

3. Start the development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## Features

- Landing page with all available plans
- User/Trainer authentication (Login/Signup)
- Trainer dashboard for creating/editing/deleting plans
- Plan details page with subscription functionality
- User feed showing plans from followed trainers and purchased plans
- Trainer profile page with follow/unfollow functionality
- Role-based UI rendering

## Tech Stack

- React 18
- React Router DOM
- Axios for API calls
- Context API for state management
